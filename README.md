# Customer Segmentation and Behavioral Insights Using ML

This project applies **Unsupervised Machine Learning** to segment customers into meaningful groups based on their purchasing and spending behavior. By using clustering techniques such as **K-Means** and **PCA for visualization**, the project uncovers behavioral patterns that can support **personalized marketing, customer retention, and business decision-making**.
